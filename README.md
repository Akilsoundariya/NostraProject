# Nostra – E-commerce Website

Nostra is a responsive e-commerce website built using **HTML, CSS, and JavaScript**.
The project includes a modern UI with product collections, filtering, search functionality, and a contact form.

## 🚀 Features

* Responsive navigation bar
* Product collections page
* Product search functionality
* Product filtering (category / arrival)
* Mobile sidebar menu
* Contact form
* Newsletter subscription section

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts
